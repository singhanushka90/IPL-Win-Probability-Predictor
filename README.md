# 🏏 About Project

The **IPL Win Probability Predictor** is a Machine Learning powered REST API that predicts the winning probability of a team during a live IPL match based on the current match situation.

The project demonstrates:

- Machine Learning
- Feature Engineering
- Data Preprocessing
- FastAPI
- REST APIs

⚙ Installation

Clone Repository

git clone https://github.com/singhanushka90/IPL-Win-Probability-Predictor.git

Move into Folder

cd IPL-Win-Probability-Predictor

Create Virtual Environment

python -m venv venv

Activate Environment

Windows

venv\Scripts\activate

Linux / Mac

source venv/bin/activate

Install Dependencies

pip install -r requirements.txt

Run FastAPI

uvicorn main:app --reload

---

📡 API Endpoint

POST /predict

Sample Input

{
  "batting_team":"Mumbai Indians",
  "bowling_team":"Chennai Super Kings",
  "city":"Mumbai",
  "runs_left":80,
  "balls_left":60,
  "wickets":6,
  "total_runs_x":180,
  "current_score":136
}

---

Sample Output

{
    "Win Probability":82.63,
    "Lose Probability":17.37
}
